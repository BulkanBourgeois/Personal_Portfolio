# Notes for agents

## Commands

- Dev server: `npm run dev`
- Typecheck: `npm run check`
- Build: `npm run build`
- E2E tests: `npm test`

## Project structure (MVP)

- 3D scene entry: `src/lib/scene/Scene.svelte`
- UI overlay entry: `src/lib/ui/overlay/TVOverlay.svelte`
- Project content data: `src/lib/data/projects.ts`
- Stores: `src/lib/stores/`
- Runtime assets: `static/models/`, `static/textures/`, `static/images/`, etc.
